# Версии превью YurClick

Каждая правка деплоится на отдельный URL. Это позволяет переключаться между версиями и сравнивать их визуально.

## Соглашение

- Формат версии: `v4.11.<N>` (по основному ТЗ v4.11)
- Каждая новая версия = отдельный коммит + отдельный URL деплоя
- Активной считается **последняя** в списке
- Старые версии остаются доступными по своим URL

## Текущая активная версия

**v4.11.16** — Перевод таблицы клиентов в режим аккордеона (один клиент = один tbody, рамка indigo вокруг 3 строк), вкладки стадии «Подготовка документов» / «Поданы в суд» со счётчиками, кнопка «Есть неточности» заменяет статус + Drawer с прогрессом, столбец «Чат» с бейджем непрочитанных, кнопка быстрой загрузки на всю ширину

## Как откатиться к предыдущей версии

Сообщить агенту: «откати к v4.11.X». Текущее состояние перезапишется состоянием указанной версии.

## Как сравнить две версии

Открыть оба URL в соседних вкладках браузера. URL остаются рабочими бессрочно.
